# To-Do-List
This is Simple To Do List made with using:
  * Node.js
  * Express.js
  * MongoDB
  
 ### Where you can add new task or delete completed task .
 ### Make your own custom list.
 
 ## Routes
  * Home Page = /
  * Compose custom list as per you(i.e : School, work) = /School or /work
